# Employee-data-anaysis
Employee Data analysis project using Excel, SQL, and Power BI

## 📌 Project Overview
This project analyzes employee data to identify workforce trends, salary distribution, attrition patterns, and departmental insights. The objective is to provide data-driven insights to improve HR decision-making.

## 🛠 Tools Used
- Microsoft Excel
- SQL
- Python (Pandas, Matplotlib)
- Power BI (Dashboard Visualization)

## 📊 Key Analysis Performed
- Calculated overall attrition rate and department-wise attrition
- Analyzed salary distribution across departments
- Evaluated gender diversity metrics
- Identified relationship between experience and salary
- Created interactive HR dashboard

## 📈 Key Insights
- Overall attrition rate: 18%
- IT department showed highest employee turnover
- Average salary increased by 12% with every 3 years of experience
- Sales department had highest workforce strength

## 📂 Project Files
- Raw Dataset (XLSX)
- Data Cleaning Script (Python)
- SQL Queries
- Power BI Dashboard
- Final Report

## 📎 Conclusion
The analysis provides actionable HR insights that help improve employee retention strategies and workforce planning.
